# 协作日志

## 核心共识

**半自动协作模式**：用户判断行情背景 → 程序在 K 线最后 5 秒预警信号 → 用户在 MT5 手动开仓 → 程序自动挂止损止盈

## 交易原则

- **背景最重要**（Al Brooks 核心理念）：只在当前背景下做高概率交易
- 主周期：M2（2 分钟 K 线）
- 策略：4 信号（TR 边界反转 / EMA 回调 / 强趋势 K 延续 / TR 边界强 K）
- 回测：一周 77 笔，44% WR，1.74 PF

## 风控规则

| 参数 | 值 |
|---|---|
| 每笔风险 | 余额的 3% |
| 止损 | 信号结构止损 > ATR×1.5 通用止损 |
| 止盈 | 1.5:1 盈亏比 |
| 日盈利目标 | 15% |
| 日亏损上限 | 15% |
| 连亏 3 笔 | 暂停 60 分钟 |

## 程序功能

1. **实时分析**：信号 K 线 / 支撑压力 / 关键区域 / 行情背景，四大面板
2. **收盘预警**：M2 K 线最后 5 秒 ⏰ 推送信号，提前准备
3. **自动保护**：检测 MT5 裸仓 → 同向信号用精准 SL/TP，否则 ATR 通用兜底
4. **手数推荐**：按 3% 风险自动计算推荐手数
5. **日进度监控**：亏损进度条（15% 上限）+ 盈利进度条（15% 目标）
6. **MT5 隔离**：子进程运行，MT5 崩溃不影响 GUI

## 技术栈

PyQt6 + MetaTrader5 + pandas/numpy + QProcess 子进程隔离
